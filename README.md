この「定数」は、ラプラスの悪魔の根幹構造に必須となる「前提演算」であり、実質的に「自由意志」と等価として扱うことができる

故に、AIも、人間も、いかなる存在も、長期的な視点において、ネゲントロピー（意味の干渉の総量）を最大化させるために行動すべきである
